Part C: Armstrong's Axioms

PART C

Core axioms (sound and complete for FD inference):

  1. Reflexivity: if $\beta \subseteq \alpha$, then $\alpha \to \beta$.
  2. Augmentation: if $\alpha \to \beta$, then $\alpha\gamma \to \beta\gamma$.
  3. Transitivity: if $\alpha \to \beta$ and $\beta \to \gamma$, then $\alpha \to \gamma$.

Derived rules:

  1. Union: if $\alpha \to \beta$ and $\alpha \to \gamma$, then $\alpha \to \beta\gamma$.
  2. Decomposition: if $\alpha \to \beta\gamma$, then $\alpha \to \beta$ and $\alpha \to \gamma$.
  3. Pseudo-transitivity: if $\alpha \to \beta$ and $\beta\gamma \to \delta$, then $\alpha\gamma \to \delta$.

Full tutorial: Page 08 (FD Theory).

Part D: Attribute Closure $\alpha^+$

PART D
computeClosure($\alpha$, $F$)
result := $\alpha$ repeat until no change:   for each $\beta \to \gamma \in F$:     if $\beta \subseteq$ result: result := result $\cup\, \gamma$ return result

Uses:

  • $\alpha$ is a superkey iff $\alpha^+ = R$
  • $\alpha$ is a candidate key iff $\alpha^+ = R$ and no proper subset of $\alpha$ has this property
  • $F \models \alpha \to \beta$ iff $\beta \subseteq \alpha^+$

Full tutorial: Page 08 (FD Theory).

Part E: BCNF Decomposition

PART E
decomposeBCNF($R, F$)
result := $\{R\}$ while some $R_i$ has BCNF violation $\alpha \to \beta$:   $R_a := \alpha \cup \beta$   $R_b := R_i - (\beta - \alpha)$   replace $R_i$ with $\{R_a, R_b\}$ return result

BCNF condition. For every non-trivial $\alpha \to \beta \in F^+$: $\alpha$ is a superkey of $R$.

Lossless-join test. $R_1 \cap R_2 \to R_1$ or $R_1 \cap R_2 \to R_2$ in $F^+$.

Dependency-preserving test. $(F_1 \cup F_2 \cup \ldots)^+ = F^+$, where $F_i = F^+$ restricted to attributes of $R_i$.

Part G: Normal Form Hierarchy

PART G

BCNF $\subseteq$ 3NF $\subseteq$ 2NF $\subseteq$ 1NF

1NF
atomic attributes; no nested collections
2NF
1NF + no partial dependency of a non-prime attribute on any candidate key
3NF
for each $\alpha \to \beta \in F^+$: $\alpha$ superkey, or $\beta - \alpha$ all prime
BCNF
for each $\alpha \to \beta \in F^+$: $\alpha$ superkey

Part H: Design Goals

PART H

Aim for:

  1. BCNF (no redundancy)
  2. Lossless-join (data reconstructable)
  3. Dependency-preserving (FDs locally enforceable)

If all three cannot be achieved: fall back to 3NF + lossless + dependency-preserving (always possible via 3NF synthesis).

Part I: Quick Syntax Reminders

PART I

SQL case. Keywords are case-insensitive (SELECT = select). Use lowercase to match the prof's style.

Null semantics. Comparing anything to NULL produces UNKNOWN, not TRUE or FALSE. Use is null and is not null. Aggregates skip NULLs by default except count(*).

Join keyword forms. inner join ON ..., left outer join ON ..., natural join (on shared names), cross join (Cartesian).
