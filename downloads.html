Suggested Study Order

  1. Start with the Complete Study Guide PDF to build up every topic from the ground up. Use it alongside Pages 01-10 on the site for interactive walkthroughs.
  2. Work through the Practice Final Exam PDF on paper, closed-book, 180 minutes, pen only. Treat it like the real thing.
  3. Mark yourself using the Solutions PDF, paying attention to the "Mirrors" line on every question that maps the practice question back to the assignment or midterm pattern it echoes.
  4. On exam morning, skim the Condensed Solutions PDF; it is the "my-style" pattern pack; one page per question; pattern first, answer shape, memory hook. Use it as the last thing you read before walking into the exam room.
  5. Revisit any walkthrough page on the site for topics you missed; every walkthrough now shows the handwritten paper-layout of the answer.

Pro tip for the practice final

Print the practice exam at least 24 hours before exam day. Sit down, phone away, 3-hour timer, pen only. If you can finish it with 15+ minutes left, you're ready. If you're missing questions in the same category (usually normalization or RA division), spend a focused study block on that walkthrough page the next morning.
